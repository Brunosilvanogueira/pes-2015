# PES Spirit 2015 (Prototype Jogável)

Este repositório contém um jogo de futebol jogável no navegador, inspirado na sensação de controle e ritmo do PES clássico.

## Como jogar agora
1. Abra `index.html` no navegador.
2. Jogue imediatamente (sem dependências).

## Controles
- `Setas` (ou `WASD`): direção do jogador controlado
- `Shift`: sprint
- `J`: passe / bote
- `K`: chute
- `R`: reiniciar partida

## Regras de controle automático
- Você controla automaticamente o jogador do seu time que está com a bola.
- Se seu time não estiver com a posse, você controla automaticamente o jogador mais próximo da bola.

## Implementado nesta versão
- Partida 11 vs 11 completa em campo 2D.
- Jogadores visíveis com numeração em campo.
- Física de bola, posse, passe, chute e desarme.
- IA adversária com estilo compacto.
- Sistema simples de stamina e emoção.
- HUD com placar, tempo, ação recente e atleta controlado.
